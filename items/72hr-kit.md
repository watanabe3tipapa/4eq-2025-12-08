| 予備電池（単三・単四等） | 数セット | 機器に合わせて |
| ラジオ（AM対応・手回しまたは電池式） | 1台 | 災害情報受信用 |

| 断熱マット／段ボール | 1枚 | 床からの冷気対策 |
| 保温着（ダウン等） | 上下各1 | 重ね着用に複数薄手も |
| 予備下着・靴下 | 2–3組 | 濡れ対策で替えを多めに |
| 使い捨てカイロ | 10–20個 | ポイント保温（胸・背中） |

| 生理用品・オムツ | 必要量 | 家族構成に応じて |

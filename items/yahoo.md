# Yahoo! の主要防災サービス

> アプリ登録を前提としたサービスなので一覧表ではリンク切れが多い。このサービスと機能について参考までに提示します。

| サービス | 主な機能 | 公式リンク |
|---|:---|---|
| Yahoo!防災速報（アプリ・ウェブ） | 緊急地震速報、津波・洪水・土砂災害・大雨・台風・熱中症等のプッシュ通知／現在地＋国内3地点設定／リアルタイム震度マップ／災害マップ／ユーザー投稿共有／通知細目設定 | https://emg.yahoo.co.jp/ |
| 災害マップ（Yahoo!防災内） | 避難所・給水所・支援拠点表示、浸水想定、通行止め、ユーザー投稿反映 | https://emg.yahoo.co.jp/map/ |
| 防災手帳 / 防災タイムライン | 避難場所登録、備蓄チェックリスト、世帯別備蓄量算出、行動タイミング通知 | https://emg.yahoo.co.jp/manual/（防災手帳ページ） |
| 自治体向け情報連携（Lアラート等） | 自治体からの避難指示・避難所開設等の配信（Lアラート連携） | https://emg.yahoo.co.jp/（サービス概要内に記載） |
| ライフライン・インフラ情報 | 電気・水道・交通の供給状況、通行止め情報表示 | https://emg.yahoo.co.jp/map/（災害マップ内のレイヤー） |
| 投稿・共有機能 | ユーザーの写真・現地情報投稿を災害マップへ反映（誤情報抑制） | https://emg.yahoo.co.jp/map/ |
| 通知／アラート設定 | 災害種別ごとのON/OFF、震度・雨量閾値、夜間サイレント等の詳細設定 | https://emg.yahoo.co.jp/help/（ヘルプ／設定ガイド） |
| Yahoo!天気・災害連携 | 気象情報、雨雲レーダー、災害関連記事や特集の提供 | https://weather.yahoo.co.jp/weather/ |
| 防災関連コンテンツ | 避難行動ガイド、緊急連絡先、備蓄リスト、訓練情報 | https://emg.yahoo.co.jp/feature/ |

---

> 主要操作（通知設定・避難所確認・防災手帳の使い方）は以下のとおり。

| 操作 | 短い手順 |
|---|---|
| 通知（緊急地震速報・気象等）の設定 | 1. アプリを開く→右下「設定」または歯車アイコン 2. 「通知設定」を選択 3. 災害種別（地震・津波・大雨・熱中症等）をON/OFF、震度・雨量の閾値を設定 4. 現在地＋追加地点（最大3地点）を登録 5. 夜間サイレントなど優先度を必要に応じて調整 |
| 避難所・災害マップ確認 | 1. アプリ／ウェブの「災害マップ」へ 2. 地図上で現在地や検索で住所入力 3. レイヤ（避難所・給水・浸水想定・通行止め）を必要に応じて表示 4. 避難所をタップして開設状況や収容情報を確認 |
| 防災手帳（避難計画・備蓄管理）の利用 | 1. アプリの「防災手帳」または「タイムライン」を開く 2. 世帯情報（人数・年齢・ペット等）を入力 3. 避難場所・集合場所を登録 4. 備蓄リストを確認・所持品をチェックして備蓄量を更新 5. 「行動タイミング通知」をONにして発災時のガイドを受け取る |
| ユーザー投稿（現地情報）の共有 | 1. 災害マップ上の「投稿」またはカメラアイコンを選択 2. 写真・状況を入力して投稿（位置情報ON） 3. 投稿はAI／人のチェック後マップに反映される |
| 自治体情報（Lアラート）の受信確認 | 1. 設定→「自治体情報受信」または「Lアラート」 2. 居住地の自治体が選択されているか確認 3. 避難指示等のメッセージ受信をONにする |
